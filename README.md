# ShiroPlus
这里是ShiroPlus，由Shiro演变而来，提供更多功能和优化。以及一些美化。

> [!IMPORTANT]
> ShiroPlus仍在早期开发中，请不要随意部署到生产环境。
> ShiroPlus由Shiro最新版本演变而来，Shiro最新版本并不稳定，所以ShiroPlus也不稳定，我们正在努力修复Bug。
> 我们并不推荐您使用ShiroPlus，而是建议您使用[Shiro](https://github.com/Innei/Shiro)或者是[余白 / Yohaku](https://github.com/Innei/Yohaku)并搭配官方推荐的V10后端（yohaku推荐最新版本）。
>与Shiro不同， **ShiroPlus 要求 Mix Space Core 版本 >= 11.x**，低版本可能会出现兼容性问题（如在V10后端不能处理在线人数）。

一个极简主义的个人网站主题，如纸的纯净，似雪的清新。

专为 [Mix Space](https://github.com/mx-space) 生态系统设计的现代化个人站点前端。
